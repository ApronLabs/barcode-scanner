# 매출지킴이 바코드 스캐너

ASUS 노트북용 바코드 스캐너 프로그램

## 기능

- **입고 모드**: 바코드 스캔 시 재고 +N
- **출고 모드**: 바코드 스캔 시 재고 -N
- **수량 조절**: 한 번에 여러 개 입출고 가능
- **Tab 키**: 입고/출고 모드 빠른 전환

## 설치 방법 (Windows)

### 1. Node.js 설치
1. https://nodejs.org 접속
2. **LTS 버전** 다운로드 (예: 20.x.x LTS)
3. 설치 파일 실행 → 기본 설정으로 설치

### 2. 프로그램 설치
1. 이 폴더(`barcode-scanner`)를 ASUS 노트북에 복사
2. 폴더 안에서 **Shift + 우클릭** → **여기에 PowerShell 창 열기**
3. 아래 명령어 실행:
```powershell
npm install
```

### 3. 프로그램 실행
```powershell
npm start
```

또는 `start.bat` 파일 더블클릭

### 4. 브라우저에서 접속
- 브라우저(Chrome 권장) 열기
- 주소창에 입력: `http://localhost:3333`

## 사용 방법

### 입고 (기본)
1. **입고** 버튼이 활성화된 상태에서 스캔
2. 바코드 스캔 → 재고 +1 증가

### 출고
1. **출고** 버튼 클릭 (또는 Tab 키)
2. 화면이 빨간색으로 변경됨
3. 바코드 스캔 → 재고 -1 감소

### 수량 조절
- −/+ 버튼으로 수량 조절 (1~99)
- 스캔 시 설정된 수량만큼 입출고

### 단축키
- **Tab**: 입고 ↔ 출고 모드 전환
- **Enter**: 바코드 수동 입력 확인

## 바로가기 만들기 (선택)

매번 명령어 치기 귀찮으면:

1. 바탕화면에 새 텍스트 파일 생성
2. 아래 내용 입력:
```batch
@echo off
cd /d "C:\경로\barcode-scanner"
npm start
pause
```
3. 파일 저장 후 확장자를 `.bat`으로 변경 (예: `스캐너실행.bat`)
4. 더블클릭으로 실행

## 문제 해결

### "npm을 찾을 수 없습니다"
- Node.js 설치 후 **컴퓨터 재시작**

### 바코드 스캔이 안됨
- 스캐너가 키보드 모드인지 확인 (대부분 기본값)
- 브라우저 입력창에 커서가 있는지 확인

### "등록되지 않은 바코드입니다"
- 해당 품목에 바코드가 등록되어 있는지 확인
- 매출지킴이 웹에서 품목 바코드 등록 필요

### "재고가 부족합니다"
- 출고 모드에서 현재 재고보다 많이 출고하려 할 때 발생
- 현재 재고 확인 후 다시 시도
