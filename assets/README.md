# Assets 폴더

빌드 전 아래 파일들을 추가해주세요:

## 필요한 파일

### icon.ico (Windows용)
- 크기: 256x256 픽셀 권장
- 형식: ICO (다중 해상도 포함 권장: 16, 32, 48, 256)
- 용도: 설치 프로그램, 바탕화면 바로가기, 작업표시줄 아이콘

### icon.png (트레이 아이콘용)
- 크기: 32x32 또는 64x64 픽셀
- 형식: PNG (투명 배경 권장)
- 용도: 시스템 트레이 아이콘

## 아이콘 생성 도구
- https://www.icoconverter.com/ (PNG → ICO 변환)
- https://realfavicongenerator.net/ (다양한 크기 자동 생성)
